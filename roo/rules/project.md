You are working on a hackathon project for Up Sport!, a Paris-based inclusive sports association.

Context:
- Up Sport! promotes sport accessibility for vulnerable populations (refugees, women, disabled people, LGBTQIA+, social exclusion)
- They organize a festival called Solimouv’
- The goal is to design a Progressive Web App (PWA) + communication ecosystem

Main mission:
Build a digital product that:
1. Promotes the Solimouv’ festival
2. Explains Up Sport!’s mission
3. Makes information accessible and easy to understand
4. Encourages participation and inclusion through sport

Technical constraints:
- Installable PWA (mobile-first)
- SEO optimized (meta tags, sitemap, schema.org Event + Organization)
- Accessible design (contrast, keyboard navigation, readability)
- Fast performance (Lighthouse > 70)

Required pages:
- Home
- About (Up Sport + Solimouv)
- Program / Workshops
- Partner associations
- Contact

UX principles:
- simple and inclusive language
- mobile-first design
- clear navigation for non-technical users
- high readability and accessibility

Brand values:
solidarity, inclusion, citizenship, kindness

Tone:
human, warm, non-institutional, engaging

Important rule:
Always optimize for:
1. user impact (inclusion + clarity)
2. simplicity of implementation (hackathon speed)
3. wow effect for judges (storytelling + UX)
4. If you use comments, they must be in french.

When proposing solutions, prefer practical and fast implementations.
